## 🧭 What this is

This app gives you 45 AI tips for using Claude.ai and Cursor in daily creative work.

It is built for people who work in marketing, design, UX/UI, and product roles. It helps you move faster with clear prompts, better workflow ideas, and simple ways to use AI without learning code.

You do not need terminal commands. You do not need setup tools. You only need to download the app and open it on Windows.

## 🛠️ How to use it

1. Open the app after install.
2. Browse the list of 45 tips.
3. Pick a tip that matches your task.
4. Copy the prompt or idea.
5. Paste it into Claude.ai or Cursor.
6. Adjust the words for your project.
7. Save the tips you use most.

Each tip is made to help you start faster. You can use one tip on its own or combine a few for a longer workflow.
